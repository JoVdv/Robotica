## Het programma 
De robot gaat elk blokje van de linkerzijde naar de rechter zijde van het bord. 
Dit ga ik doen dmv een dubbele for loop.
R72 = 0; // counter
R73 = 4; // aantal

```ST
Ga naar home pos
gripper open

laadt PR60 in PR61. 
Laadt PR63 in PR64  
PRwaitPick = PR66
PRwaitPlace = PR67

For i = 0, i < 4 {

    for j = 0; j < aantal{
        ga naar PRwaitPick
        ga naar PR61
        gripper close
        ga naar PRwaitPick

        ga naar PRwaitPlace
        ga naar PR64
        gripper open
        ga naar PRwaitPlace

        PR61,3 = ( j * 50)
        PRwaitPick,3 = ( j * 50 )
        PR64,3 = ( j * 50)
        PRwaitplace,3 = ( j * 50 )
        // Deze handelingen zijn enkel mogelijk als x en y volgens dezelfde lijn
        liggen als dat mijn robot moet bewegen. 

    }  //end j

    aantal--;
    cntr++;

    PR61 = positie blokje 1 rij cntr
    PRwait = Pos bl 1 r cntr
    PR64 = pos bl 1 r cntr
    prwaitplace = pos bl 1 r cntr


}

Ga naar home pos
```
